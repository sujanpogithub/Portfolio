Automated Multilevel Room Temperature Control System

Smart dustbin for physically disabled people

The design is for a smart waste container that uses a servo motor and IR sensor to detect users and open the lid, with the aim of assisting physically disabled users. The dustbin can be remote operated for waste disposal by RF transmitter and receiver with encoder and decoder

-The system monitors trash levels with ultrasonic sensors and sends data wirelessly to a central server. Waste authorities can access the information via a web interface to improve garbage collection efficiency.
